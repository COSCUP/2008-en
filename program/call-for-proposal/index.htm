Call for Proposal

We want your involvement! If you are involved in development, testing, or deployment of open source software and are willing to tell others your work, or want to listen to a particular topic, please contact us (mailto : 2008cfp @ coscup.org).

The range of topics this year include, but not limited to:

  • open source hand-held device
  • web development
  • open source small form factor laptops
  • firefox or other web browser
  • Java related
  • community activities
  • security
  • Chinese input method workshop
  • Other Open-Source Related Topics

Every speaker will have 20 minutes (negotiable, please e-mail), except for lightning talks which will be limited to 5 minutes.
